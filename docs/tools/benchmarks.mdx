## Coverage

<Info>
  SafeMode includes **every installable CLI tool** from both leaderboards. Tools that are not included are either:
  - GUI-only applications (Warp, Capy, DeepAgent Desktop)
  - Internal/proprietary with no public release (Ante, OB-1, Engine Labs)
  - Research harnesses that aren't standalone CLIs (Terminus-KIRA, TongAgents)
  - Require Docker-in-Docker at runtime (SWE-agent, OpenHands)
</Info>
