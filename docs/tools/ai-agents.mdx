# AI Coding Agents

Every agent is pre-installed and available from the terminal inside the container. Launch them from the web dashboard or via `xterm` in the noVNC desktop.

## Tier 1 — Top Benchmark Performers

These tools rank highest on [terminal-bench](https://www.tbench.ai), the Stanford x Laude benchmark for terminal coding agents.

<AccordionGroup>
  <Accordion title="ForgeCode — #1 on terminal-bench v2.0 (81.8%)" icon="hammer">
    **Command:** `forge`
    **Provider:** TailCall
    **Install:** `curl -fsSL https://forgecode.dev/cli | sh`

    Multi-model AI coding agent with a beautiful TUI. Supports 300+ models across Claude, GPT, Gemini, Grok, DeepSeek, and more. Three modes: interactive TUI, one-shot CLI, and ZSH plugin.

    [Documentation](https://forgecode.dev) | [GitHub](https://github.com/tailcallhq/forgecode)
  </Accordion>

  <Accordion title="Droid — #6 on terminal-bench v2.0 (77.3%)" icon="robot">
    **Command:** `droid`
    **Provider:** Factory AI
    **Install:** `curl -fsSL https://app.factory.ai/cli | sh`

    Factory AI's terminal-based coding agent. Consistent top performer across both v1.0 and v2.0 benchmarks.

    [Documentation](https://factory.ai)
  </Accordion>

  <Accordion title="Apex2 — #1 on terminal-bench v1.0 (64.5%)" icon="star">
    **Provider:** heartyguy (research)
    **Type:** Research agent (Python)

    Achieved state-of-the-art performance through strategic simplification. Built as a weekend project, demonstrates that architecture beats complexity.

    [GitHub](https://github.com/heartyguy/Apex2-Terminal-Bench-Agent)
  </Accordion>
</AccordionGroup>

## Open Source Agents

<AccordionGroup>
  <Accordion title="Goose (Block)" icon="feather">
    **Command:** `goose`
    **Stars:** 41K

    MCP-native coding agent with 70+ extensions. Block (Square) built it; Stripe forked it for their internal Minions system.

    [GitHub](https://github.com/block/goose)
  </Accordion>

  <Accordion title="Aider" icon="users">
    **Command:** `aider`
    **Stars:** 30K+
    **Install:** `pip install aider-chat`

    Most popular open-source AI pair programmer. Edits files directly in your git repo via chat. Git-native — understands your repo history.

    [Documentation](https://aider.chat) | [GitHub](https://github.com/Aider-AI/aider)
  </Accordion>

  <Accordion title="Crush CLI (Charmbracelet)" icon="heart">

  <Accordion title="Amp (Sourcegraph)" icon="bolt">
