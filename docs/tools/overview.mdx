## How Tools Are Installed

Tools are installed using their native package managers to ensure you always get the latest versions at build time:

- **npm** — Claude Code, Gemini, Codex, Copilot, Amp, Qwen, Continue, Letta, iFlow, Qoder, Cline, OpenClaw
- **pip** — Aider
- **apt** — Crush (via Charmbracelet repo)
- **curl/binary** — ForgeCode, Goose, Droid, OpenCode, Plandex, Kiro, Junie, code-server, File Browser
- **deb** — Amazon Q, Chaterm
