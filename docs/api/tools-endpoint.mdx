# GET /api/tools.json

Returns a complete manifest of all tools installed in the container, including metadata for programmatic discovery.

## Request

```bash
curl -u devuser:$PASSWORD http://your-server:8888/api/tools.json
```

## Tool Object Schema

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier |
| `name` | string | Display name |
| `description` | string | One-line description |
| `category` | string | `"ai"`, `"editor"`, or `"utility"` |
| `type` | string | `"cli"` (terminal) or `"web"` (browser-accessible) |
| `port` | number \| null | Port number for web tools, null for CLI tools |
| `url` | string \| null | URL path for web tools (e.g., `/code-server/`) |
| `command` | string \| null | Shell command to invoke the tool |
| `docs` | string | Documentation URL |

## Filtering by Category

The API returns all tools. Filter client-side:

```bash
# Get only AI agents
curl -s -u devuser:$PASSWORD http://localhost:8888/api/tools.json \
  | jq '.tools[] | select(.category == "ai") | .name'

# Get only web services
curl -s -u devuser:$PASSWORD http://localhost:8888/api/tools.json \
  | jq '.tools[] | select(.type == "web") | {name, url, port}'

# Count tools
curl -s -u devuser:$PASSWORD http://localhost:8888/api/tools.json \
  | jq '.tools | length'
```

## Example: Agent Discovery

An AI agent can use this endpoint to discover what tools are available and how to invoke them:

```python
import requests

resp = requests.get(
    "http://server:8888/api/tools.json",
    auth=("devuser", "password")
)
tools = resp.json()["tools"]

# Find all CLI tools
cli_tools = [t for t in tools if t["type"] == "cli"]

# Get the command for Claude Code
claude = next(t for t in tools if t["id"] == "claude-code")
print(f"Run: {claude['command']}")  # "claude"
```
