# API Overview

SafeMode exposes a JSON API for programmatic tool discovery. This is designed for AI agent orchestration, CI/CD pipelines, and automation scripts.

## Base URL

```
http://your-server:8888/api/
```

All API endpoints require HTTP Basic Auth (`devuser` / your password).

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/tools.json` | Complete tool manifest with metadata |

## Authentication

```bash
curl -u devuser:$PASSWORD http://your-server:8888/api/tools.json
```

## CORS

The API includes permissive CORS headers (`Access-Control-Allow-Origin: *`) for cross-origin access from browser-based agents.
